    В данном приложение отображаются точки в декартовой системе координат. Точки генерируются рандомно в отдельном потоке.
Для написания проекта испльзовался Qt 5.15.2 в среде Ubuntu 20.10. Для построения системы координат использовалась статическая библиотека
Grapher2D. Данная библиотека плод собственного творения, который был издан в 2016 году.
   В каталоге имеется исполняемый файл, который запускается скриптом testing_work.sh в директории export/Distributionkit.
